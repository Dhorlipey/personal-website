Numerical Linear Algebra · Technical Report
QR Decomposition via Gram–Schmidt
Dolapo Onifade · Department of Mathematics, LSU
This report details the QR decomposition via Gram–Schmidt orthogonalization, contrasting the numerical stability of the classical and modified algorithms in finite-precision arithmetic. The enhanced stability of the modified Gram–Schmidt algorithm is demonstrated analytically and numerically. The report then applies QR decomposition to linear least-squares problems, comparing its numerical robustness against the normal equations approach, particularly in the context of matrix ill-conditioning.

Machine Learning · Group Project · CSC 7333
Sentiment Analysis of Text Reviews Using Classical and Deep Learning Methods
Basit Abdulfatai, Marvellous Mareya, Solomon Joe, Dolapo Onifade, Victory Oghale, Philip John
This project trains and evaluates four sentiment classifiers on the IMDb Movie Reviews dataset (50,000 reviews), comparing classical approaches (Naïve Bayes, SVM) against deep learning methods (Bidirectional LSTM, fine-tuned BERT). All models undergo systematic hyperparameter search. The key finding is that a well-tuned SVM achieves the highest accuracy (90.16%) in just 25 seconds on CPU — outperforming BERT (89.68%), which requires approximately 474× more compute. The results challenge the common assumption that greater model complexity automatically yields better performance.
